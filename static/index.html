How to get a session token (click to expand)
  1. Open the same scheduler URL you pick below: public or www Texas DPS scheduler.
  2. Complete the captcha and sign in until your info loads.
  3. Chrome / Edge: F12 → Network → filter by apptapi.
  4. Open any request after login (e.g. Eligibility, Booking, or not the first auth if Authorization was empty there).
  5. Under Request headers, copy the full Authorization value (or use the token from the auth response Preview tab).
  6. Paste into Session token here → Save settings → Test connection before starting.

Tokens expire; if you see 401 later, copy a fresh one. This tool is unofficial and not affiliated with Texas DPS.
